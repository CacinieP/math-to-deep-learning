# 神经网络基础

> 一个神经网络，无论多深多复杂，数学上就是**一串复合函数**：$\hat{y} = (f_L \circ \cdots \circ f_1)(x)$，其中每一层是一个**仿射映射 + 非线性激活**。理解了"函数复合"和"线性映射"，你就理解了神经网络的全部前向结构。

**难度**：[基础]（需要高中函数概念 + 矩阵乘法 + 一点 Python）

## 一、纯数学版：神经网络是个什么函数

### 1.1 一层的数学定义

神经网络的**一层** $f_l: \mathbb{R}^{d_{l-1}} \to \mathbb{R}^{d_l}$ 定义为：

$$f_l(\mathbf{h}_{l-1}) = \sigma(W_l \mathbf{h}_{l-1} + \mathbf{b}_l)$$

其中：
- $\mathbf{h}_{l-1} \in \mathbb{R}^{d_{l-1}}$ 是上一层的输出（输入信号）
- $W_l \in \mathbb{R}^{d_l \times d_{l-1}}$ 是**权重矩阵**（线性变换）
- $\mathbf{b}_l \in \mathbb{R}^{d_l}$ 是**偏置向量**（平移）
- $\sigma: \mathbb{R} \to \mathbb{R}$ 是**激活函数**，按元素作用（引入非线性）

**拆解**：一层 = 仿射变换（$W\mathbf{h}+\mathbf{b}$）+ 逐元素非线性（$\sigma$）。

> 📖 矩阵作为线性映射：[[Mathematics-Universe/04-线性代数/02-矩阵/矩阵详解.md]]
> 📖 仿射变换：[[Mathematics-Universe/04-线性代数/03-向量与向量空间/向量与向量空间详解.md]]

### 1.2 整个网络 = 函数复合

$L$ 层网络的整体映射是这些层的**复合**：

$$\hat{\mathbf{y}} = f_L \circ f_{L-1} \circ \cdots \circ f_1(\mathbf{x})$$

展开写：

$$\hat{\mathbf{y}} = \sigma(W_L(\sigma(W_{L-1}(\cdots \sigma(W_1 \mathbf{x} + \mathbf{b}_1)\cdots) + \mathbf{b}_{L-1})) + \mathbf{b}_L)$$

**关键洞察**：如果去掉所有 $\sigma$，网络退化成纯线性映射的复合——而**线性映射的复合还是线性映射**：

$$W_L W_{L-1} \cdots W_1 = W_{\text{总}}$$

那么再深的网络也等价于单层。**激活函数是深度的唯一来源**。

### 1.3 为什么需要非线性：一个反例

考虑两层无激活的网络：$f(\mathbf{x}) = W_2(W_1 \mathbf{x}) = (W_2 W_1)\mathbf{x}$。

令 $W_{\text{eff}} = W_2 W_1$，则 $f(\mathbf{x}) = W_{\text{eff}} \mathbf{x}$ —— 与一层无异。加任意多层都不增加表达能力。

**结论**：非线性激活 $\sigma$ 不是装饰，是深度网络有意义的**数学前提**。

## 二、应用映射层：每一部分的几何直觉

### 2.1 权重矩阵 = 坐标系的旋转/拉伸/压缩

$W\mathbf{h}$ 在几何上是对向量 $\mathbf{h}$ 做**线性变换**：
- 旋转（正交矩阵）
- 拉伸/压缩（对角矩阵）
- 投影（秩亏损矩阵）
- 剪切（一般可逆矩阵）

```
输入空间 R^d                 特征空间 R^h
   ●  ●                         ●
  ● ●  ●     ──W: d→h──▶      ● ●
   ●  ●                        ● ●
 (原始像素)                  (旋转+压缩后的特征)
```

**意义**：每一层把数据投影到一个新的"视角"，让后一层更容易分类。

### 2.2 偏置 = 平移原点

$\mathbf{b}$ 的作用是**移动决策边界**。没有偏置，决策面必须过原点，表达能力受限。

**几何**：仿射变换 $=$ 线性变换 $+$ 平移，能表示不过原点的任意超平面。

### 2.3 激活函数 = 弯曲空间

| 激活 | 公式 | 形状 | 特点 |
|------|------|------|------|
| Sigmoid | $\frac{1}{1+e^{-x}}$ | S 形，值域 $(0,1)$ | 易饱和，梯度消失 |
| Tanh | $\frac{e^x - e^{-x}}{e^x + e^{-x}}$ | S 形，值域 $(-1,1)$ | 零中心，仍会饱和 |
| ReLU | $\max(0, x)$ | 折线 | 计算快，缓解梯度消失 |
| Leaky ReLU | $\max(0.01x, x)$ | 微斜折线 | 解决 ReLU 死亡 |

**ReLU 的非线性**：虽分段线性，但组合后在整条实轴上是非线性的（折点处不可导，但次梯度存在）。

## 三、前向传播：从输入到输出的完整数据流

### 3.1 一个 3 层网络

```
输入 x ∈ R^784 (MNIST 展平)
    │
    ▼  f₁: W₁(128×784), b₁(128), ReLU
h₁ ∈ R^128
    │
    ▼  f₂: W₂(64×128), b₂(64), ReLU
h₂ ∈ R^64
    │
    ▼  f₃: W₃(10×64), b₃(10), (softmax)
ŷ ∈ R^10 (各类概率)
```

### 3.2 维度流的关键规则

相邻层的维度必须匹配：第 $l$ 层的输入维度 $d_{l-1}$ 必须 $=$ $W_l$ 的列数。

$$\underbrace{W_l}_{d_l \times d_{l-1}} \cdot \underbrace{\mathbf{h}_{l-1}}_{d_{l-1}} + \underbrace{\mathbf{b}_l}_{d_l} = \underbrace{\mathbf{z}_l}_{d_l} \xrightarrow{\sigma} \underbrace{\mathbf{h}_l}_{d_l}$$

**口诀**：前层输出维度 $=$ 本层权重列数。

## 四、工程实现层：PyTorch 最小可运行示例

### 4.1 手写一个 MLP 前向传播

```python
import torch
import torch.nn.functional as F

class MLP(torch.nn.Module):
    def __init__(self, in_dim=784, hidden=128, out_dim=10):
        super().__init__()
        # 三层：784 → 128 → 64 → 10
        self.fc1 = torch.nn.Linear(in_dim, hidden)    # W₁: 128×784, b₁: 128
        self.fc2 = torch.nn.Linear(hidden, 64)        # W₂: 64×128,  b₂: 64
        self.fc3 = torch.nn.Linear(64, out_dim)       # W₃: 10×64,   b₃: 10

    def forward(self, x):
        h1 = F.relu(self.fc1(x))   # f₁: 仿射 + ReLU
        h2 = F.relu(self.fc2(h1))  # f₂: 仿射 + ReLU
        logits = self.fc3(h2)      # f₃: 仿射(softmax 在损失里算)
        return logits

model = MLP()
print(model)
# 输出维度检查
x = torch.randn(32, 784)   # batch=32 的 MNIST
y = model(x)
print(y.shape)             # torch.Size([32, 10])
```

### 4.2 `nn.Linear` 内部就是 $W\mathbf{x}+\mathbf{b}$

```python
layer = torch.nn.Linear(784, 128)
print(layer.weight.shape)  # torch.Size([128, 784]) = W
print(layer.bias.shape)    # torch.Size([128])      = b
# 前向 = weight @ x.T + bias  (批量并行)
```

### 4.3 验证"无激活=退化"

```python
# 两个无激活的线性层
l1 = torch.nn.Linear(10, 20)
l2 = torch.nn.Linear(20, 5)
x = torch.randn(10)
# 等价于单个线性层
W_equiv = l2.weight @ l1.weight
b_equiv = l2.weight @ l1.bias + l2.bias
print(torch.allclose(l2(l1(x)), x @ W_equiv.T + b_equiv, atol=1e-6))  # True
```

## 五、参数量计算

网络的总参数量 $=$ 所有 $W$ 和 $\mathbf{b}$ 的元素总数：

| 层 | $W$ 形状 | $\mathbf{b}$ 形状 | 参数量 |
|----|---------|------------------|--------|
| fc1 | $128 \times 784$ | $128$ | $100{,}352 + 128 = 100{,}480$ |
| fc2 | $64 \times 128$ | $64$ | $8{,}192 + 64 = 8{,}256$ |
| fc3 | $10 \times 64$ | $10$ | $640 + 10 = 650$ |
| **合计** | | | **109{,}386** |

这就是 MNIST 上一个小 MLP 的参数规模。Transformer 和大模型不过是把这个乘法放大到几十亿。

## 七、直觉总结

```
神经网络 = 一串带非线性的复合函数

输入 x
  → 每层做: "换个角度看数据"(W) + "挪一下原点"(b) + "弯一下空间"(σ)
  → 逐层把"难分的数据"扭成"线性可分的数据"
输出 ŷ

深度的意义: 每多一层, 表达能力指数级增长(但有梯度消失的代价)
```

**函数复合 + 仿射变换 + 非线性激活 = 神经网络的全部前向结构。** 反向传播只是教你怎么调 $W$ 和 $\mathbf{b}$。
