# 代码实现映射

> 数学概念 → PyTorch API 对照表。看到公式不知道怎么实现？查这张表。

## 基础运算

| 数学概念 | PyTorch API | 示例 |
|---------|-------------|------|
| 矩阵乘法 $A B$ | `A @ B` 或 `torch.matmul(A, B)` | `h = x @ W` |
| 逐元素乘 $\odot$ | `*` 或 `torch.mul` | `mask * x` |
| 转置 $A^\top$ | `A.T` 或 `A.transpose(-1,-2)` | `Q @ K.T` |
| 内积 $\mathbf{x}^\top\mathbf{y}$ | `x @ y` 或 `torch.dot` | `torch.dot(x, y)` |
| 范数 $\|\mathbf{x}\|$ | `x.norm()` 或 `torch.norm` | `x.norm(dim=-1)` |
| 求和 $\sum$ | `x.sum()` | `x.sum(dim=1)` |
| 指数 $e^x$ | `torch.exp` 或 `x.exp()` | `torch.exp(logits)` |
| 对数 | `torch.log` | `probs.log()` |
