# 梯度消失/爆炸的数学根源

> 为什么深网络训不动？数学答案：**梯度是各层雅可比矩阵的连乘积**，每个雅可比的特征值稍偏离 1，连乘 $L$ 层就指数放大（爆炸）或缩小（消失）。这是矩阵乘积范数与谱半径的纯线性代数问题。

**难度**：[进阶]（需要矩阵范数 + 特征值 + 反向传播）

## 一、纯数学：梯度是雅可比连乘

### 1.1 反向传播的矩阵形式

对 $L$ 层网络 $\mathbf{h}_{l} = f_l(\mathbf{h}_{l-1})$，损失对输入的梯度：

$$\frac{\partial \mathcal{L}}{\partial \mathbf{h}_0} = \frac{\partial \mathcal{L}}{\partial \mathbf{h}_L} \cdot \prod_{l=L}^{1} J_l$$

其中 $J_l = \partial \mathbf{h}_l/\partial \mathbf{h}_{l-1}$ 是第 $l$ 层的雅可比矩阵。

**关键**：梯度是 $L$ 个矩阵的连乘积。

> 📖 反向传播：[[轴线C/01-梯度→反向传播]]

### 1.2 连乘的范数行为

由矩阵范数次可乘性：

$$\left\|\prod_{l=1}^{L} J_l\right\| \leq \prod_{l=1}^{L} \|J_l\|$$

若每层 $\|J_l\| \approx \rho > 1$，连乘后 $\approx \rho^L$——**指数爆炸**。
若 $\rho < 1$，连乘 $\approx \rho^L \to 0$——**指数消失**。

### 1.3 谱半径决定长期行为

矩阵的**谱半径** $\rho(M) = \max |\lambda_i(M)|$（最大特征值绝对值）。对 powers $M^L$：

$$\rho(M) < 1 \Rightarrow M^L \to 0, \quad \rho(M) > 1 \Rightarrow \|M^L\| \to \infty$$

**深网络梯度的生死，由各层雅可比谱半径的几何均值决定**——接近 1 才稳定。

> 📖 特征值与谱：[[Mathematics-Universe/04-线性代数/05-特征值与特征向量/特征值与特征向量详解.md]]

## 二、激活函数如何放大问题

### 2.1 Sigmoid 的灾难

Sigmoid $\sigma(x) = 1/(1+e^{-x})$ 的导数：

$$\sigma'(x) = \sigma(x)(1-\sigma(x)) \leq 0.25$$

**每层雅可比含因子 $\sigma' \leq 0.25$**。$L=10$ 层连乘 $\leq 0.25^{10} \approx 10^{-6}$——梯度近乎归零。

### 2.2 ReLU 的缓解

ReLU 导数：$x>0$ 时为 1，$x<0$ 时为 0。

$$\text{ReLU}'(x) = \mathbf{1}[x > 0]$$

- 正区：导数 1，不缩小梯度（缓解消失）
- 负区：导数 0，梯度完全阻断（"死亡 ReLU"）

**ReLU 让消失问题大幅缓解**，这是它取代 sigmoid 的主要原因。

### 2.3 权重初始化的角色

若权重 $W$ 各元素方差 $\text{Var}(w_{ij}) = \sigma_w^2$，前向激活方差每层乘以 $n_l \sigma_w^2$（$n_l$ 是宽度）。要让激活方差逐层不变，需：

$$n_l \sigma_w^2 = 1 \Rightarrow \sigma_w^2 = 1/n_l \quad \text{(Xavier)}$$

这就是 Xavier/He 初始化的数学根据——让每层雅可比谱半径接近 1。

> 📖 初始化推导：[[轴线B/03-正态分布→Xavier-He初始化]]

## 三、残差连接：数学解药

### 3.1 残差改变雅可比

残差块 $\mathbf{h}_{l+1} = \mathbf{h}_l + f_l(\mathbf{h}_l)$ 的雅可比：

$$J_{l+1} = I + \frac{\partial f_l}{\partial \mathbf{h}_l}$$

**关键项 $I$**：即使 $\partial f_l/\partial \mathbf{h}_l \to 0$，$J \to I$，谱半径 $\to 1$。梯度**有高速公路直通**。

### 3.2 残差把谱半径锁在 1 附近

无残差时 $J = \partial f/\partial \mathbf{h}$ 容易 $<1$（消失）。加 $I$ 后：

$$\rho(I + A) \approx 1 + \text{小扰动}$$

只要 $A$ 的谱小，$\rho$ 接近 1，连乘不爆不消。**这就是 ResNet 能训上百层的数学根基。**

## 四、工程诊断与治疗

### 4.1 诊断：监控梯度范数

```python
for name, p in model.named_parameters():
    if p.grad is not None:
        print(f"{name}: grad norm = {p.grad.norm().item():.4e}")
# 若深层梯度 norm << 浅层 ⇒ 消失
# 若深层梯度 norm >> 浅层 ⇒ 爆炸
```

### 4.2 治疗：四件武器

| 手段 | 解什么 | 怎么做 |
|------|--------|--------|
| ReLU/LeakyReLU | 消失 | 换激活 |
| He 初始化 | 消失/爆炸 | 调权重方差 |
| 残差连接 | 消失 | 加 $+x$ |
| 梯度裁剪 | 爆炸 | `clip_grad_norm_` |
| LayerNorm | 两者 | 归一化激活 |
| 残差 + Norm 组合 | 两者 | Transformer 标配 |

```python
# 梯度裁剪
torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
```

## 六、直觉总结

```
梯度 = 各层雅可比的连乘积
每层雅可比谱半径 ρ:
  ρ>1 ⇒ 连乘指数爆炸(梯度→inf)
  ρ<1 ⇒ 连乘指数消失(梯度→0)
  ρ≈1 ⇒ 稳定

sigmoid: σ'≤0.25 ⇒ 必然消失
ReLU:   σ'∈{0,1} ⇒ 缓解消失
残差:   J=I+A, ρ≈1 ⇒ 高速公路保梯度
```

**梯度消失/爆炸 = 雅可比连乘的谱半径病。** 解药都是让谱半径靠近 1。
