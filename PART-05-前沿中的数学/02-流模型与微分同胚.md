# 流模型与微分同胚

> 流模型（Normalizing Flow）通过一串**可逆变换**把简单分布（高斯）变成复杂分布（数据）。每个变换是**微分同胚**——光滑、可逆、且逆也光滑。可逆性让似然可精确计算，雅可比行列式决定体积变化，微分几何的**变量替换公式**是它的全部数学。

**难度**：[前沿]（需要变量替换 + 雅可比行列式 + 微分几何直觉）

## 一、纯数学：变量替换与微分同胚

### 1.1 变量替换公式

若 $\mathbf{z} \sim p(\mathbf{z})$，经可逆变换 $\mathbf{x} = f(\mathbf{z})$，则 $\mathbf{x}$ 的密度：

$$p_X(\mathbf{x}) = p_Z(f^{-1}(\mathbf{x})) \left|\det J_{f^{-1}}(\mathbf{x})\right|$$

其中 $J_{f^{-1}}$ 是逆变换的雅可比矩阵，$|\det J|$ 是**体积变化系数**。

**直觉**：变换把空间拉伸/压缩，密度要除以局部体积变化（概率守恒）。

> 📖 雅可比矩阵：[[Mathematics-Universe/03-高等数学/04-多元微分学/多元微分学详解.md]]

### 1.2 微分同胚的定义

$f: \mathbb{R}^n \to \mathbb{R}^n$ 是**微分同胚**，若：
1. $f$ 双射（可逆）
2. $f$ 光滑（无穷次可微）
3. $f^{-1}$ 也光滑

**关键**：微分同胚保维数、保拓扑（不撕裂空间）。

### 1.3 链式雅可比

多个变换复合 $f = f_n \circ \cdots \circ f_1$，雅可比行列式相乘：

$$\det J_f = \prod_{i=1}^n \det J_{f_i}$$

**这就是流模型"流"的含义**——一串变换像流体层层塑形。

---

## 二、Normalizing Flow 的结构

### 2.1 基本框架

从简单噪声 $\mathbf{z}_0 \sim \mathcal{N}(0, I)$ 出发，经 $K$ 个可逆变换：

$$\mathbf{z}_K = f_K \circ \cdots \circ f_1(\mathbf{z}_0)$$

$\mathbf{z}_K$ 的对数密度（由变量替换链式展开）：

$$\log p(\mathbf{z}_K) = \log p(\mathbf{z}_0) - \sum_{k=1}^K \log\left|\det J_{f_k}(\mathbf{z}_{k-1})\right|$$

**可精确计算似然**——这是流模型相对 GAN 的核心优势。

### 2.2 设计矛盾：表达力 vs 雅可比计算

- **任意可逆变换**：表达力强，但 $\det J$ 是 $O(n^3)$，高维不可行
- **三角变换**（如 NICE、RealNVP）：雅可比是三角矩阵，$\det$ = 对角元乘积，$O(n)$

**RealNVP 的耦合层**：把维度分成两半，一半决定另一半的仿射变换——保证可逆且雅可比三角。

### 2.3 连续流（Neural ODE）

把离散变换序列极限化成连续 ODE：

$$\frac{d\mathbf{z}(t)}{dt} = v_\theta(\mathbf{z}(t), t)$$

**瞬时变量替换公式**（Chen et al., 2018）：

$$\log p(\mathbf{z}(1)) = \log p(\mathbf{z}(0)) - \int_0^1 \nabla \cdot v_\theta(\mathbf{z}(t), t)\,dt$$

散度 $\nabla \cdot v$ 替代了离散的雅可比行列式。**这把 Normalizing Flow 与 Diffusion 的概率流 ODE 统一了**。

---

## 三、可逆性的代价与收益

### 3.1 收益：精确似然

流模型能算 $p(x)$ 的精确值，用于：
- 密度估计（异常检测）
- 信息论分析
- 贝叶斯推断

### 3.2 代价：拓扑限制

微分同胚**保拓扑**——不能把单连通区域变成有洞的。所以纯流模型难以生成多模态/复杂拓扑分布（这是 Diffusion 兴起的部分原因）。

### 3.3 表达力 vs 维度

高维下，要表达复杂分布需要极深的流或极复杂的耦合层。实践中 RealNVP、Glow 等用上百层耦合。

## 四、工程实现层

### 4.1 RealNVP 耦合层

```python
import torch
import torch.nn as nn

class CouplingLayer(nn.Module):
    def __init__(self, dim, hidden=64):
        super().__init__()
        self.d = dim // 2
        self.st = nn.Sequential(nn.Linear(self.d, hidden), nn.ReLU(), nn.Linear(hidden, 2*(dim - self.d)))
    def forward(self, x):       # x → z, 返回 z 和 log|detJ|
        x1, x2 = x[:, :self.d], x[:, self.d:]
        st = self.st(x1)
        s, t = st.chunk(2, dim=1)
        s = torch.tanh(s)
        z2 = x2 * s.exp() + t
        z = torch.cat([x1, z2], dim=1)
        logdet = s.sum(dim=1)   # 三角雅可比 ⇒ logdet = Σs
        return z, logdet
    def inverse(self, z):       # z → x (生成用)
        z1, z2 = z[:, :self.d], z[:, self.d:]
        st = self.st(z1)
        s, t = st.chunk(2, dim=1)
        s = torch.tanh(s)
        x2 = (z2 - t) * (-s).exp()
        return torch.cat([z1, x2], dim=1)
```

### 4.2 对数似然

```python
def log_prob(self, x):
    z = x
    logdet = 0
    for layer in self.layers:
        z, ld = layer(z)
        logdet = logdet + ld
    # 基础分布是标准高斯
    log_pz = -0.5 * (z ** 2).sum(dim=1) - 0.5 * z.size(1) * torch.log(torch.tensor(2 * torch.pi))
    return log_pz + logdet
```

## 六、直觉总结

```
流模型 = 一串可逆变换(微分同胚)
噪声 z₀ → 经 K 层变换 → 数据 z_K
密度守恒: p(z_K) = p(z₀)·∏|det Jₖ⁻¹|

可逆 ⇒ 精确似然(优于 GAN)
保拓扑 ⇒ 难生成复杂拓扑(输给 Diffusion)
连续极限(Neural ODE) ⇒ 与 Diffusion 统一
```

**流模型 = 参数化的微分同胚。** 数学是变量替换，工程是三角雅可比的设计。
