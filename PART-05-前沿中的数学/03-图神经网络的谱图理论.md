# 图神经网络的谱图理论

> 图神经网络（GNN）的本质是**在图上做消息传递**——每个节点聚合邻居信息。谱图理论揭示：这等价于在**图拉普拉斯矩阵的特征基**上做滤波。GNN 是图信号处理里的带通滤波器，谱聚类、ChebNet、GCN 都根植于此。

**难度**：[前沿]（需要图论 + 拉普拉斯算子 + 傅里叶分析）

## 一、纯数学：图拉普拉斯算子

### 1.1 图的表示

图 $G = (V, E)$，邻接矩阵 $A$，度矩阵 $D = \text{diag}(\sum_j A_{ij})$。

### 1.2 图拉普拉斯矩阵

$$L = D - A$$

**性质**：
- 对称半正定：$L \succeq 0$
- 对常向量 $\mathbf{1}$ 特征值为 0：$L\mathbf{1} = 0$
- 对任意 $\mathbf{x}$：$\mathbf{x}^\top L \mathbf{x} = \sum_{(i,j)\in E}(x_i - x_j)^2 \geq 0$

**意义**：$L$ 度量"信号在图上的不光滑度"——相邻节点差异越大，值越大。这是图上的**拉普拉斯算子**（连续 $\Delta = -\nabla^2$ 的离散版）。

> 📖 矩阵与谱：[[Mathematics-Universe/04-线性代数/05-特征值与特征向量/特征值与特征向量详解.md]]

### 1.3 谱分解

$$L = U \Lambda U^\top, \quad \Lambda = \text{diag}(\lambda_1=0 \leq \lambda_2 \leq \ldots \leq \lambda_n)$$

- $\lambda_1 = 0$ 对应常向量（全图平均）
- $\lambda_2$（代数连通度/Fiedler 值）反映图连通性
- $\lambda_i$ 大 → 对应特征向量在邻域间剧烈振荡（高频）

**这就是图上的"傅里叶基"**——$U$ 的列向量是图频率的正交基。

## 二、图上的傅里叶变换

### 2.1 图傅里叶变换

把连续傅里叶的定义"用拉普拉斯特征基展开"推广到图：

$$\hat{x} = U^\top x \quad \text{(图傅里叶变换)}$$
$$x = U\hat{x} \quad \text{(逆变换)}$$

**$\hat{x}_i$ 是信号 $x$ 在"频率" $\lambda_i$ 上的分量**。

### 2.2 谱滤波

在频域乘滤波器 $g(\Lambda)$，再变回：

$$y = U\,g(\Lambda)\,U^\top x$$

**这就是图上的卷积**。但直接做要 $O(n^3)$ 特征分解 + $O(n^2)$ 矩阵乘，不可扩展。

### 2.3 多项式逼近（ChebNet, Defferrard 2016）

用切比雪夫多项式 $T_k$ 逼近 $g$：

$$g(\Lambda) \approx \sum_{k=0}^{K} \theta_k T_k(\tilde\Lambda)$$

**优势**：避免特征分解，只做 $K$ 次稀疏矩阵-向量乘（$O(K|E|)$）。这是 ChebNet 的核心。

### 2.4 一阶近似（GCN, Kipf 2017）

取 $K=1$，ChebNet 退化成 **GCN**：

$$H^{(l+1)} = \sigma\left(\tilde D^{-1/2}\tilde A \tilde D^{-1/2} H^{(l)} W^{(l)}\right)$$

$\tilde A = A + I$（加自环）。**GCN = 一阶切比雪夫滤波器 + 权重变换**。

## 三、消息传递的统一视角

### 3.1 消息传递框架（MPNN）

几乎所有 GNN 都能写成：

$$h_v^{(l+1)} = \phi\!\left(h_v^{(l)}, \bigoplus_{u \in \mathcal{N}(v)} \psi(h_v^{(l)}, h_u^{(l)}, e_{vu})\right)$$

- $\psi$：消息函数（邻居给当前节点的信息）
- $\bigoplus$：聚合（sum/mean/max）
- $\phi$：更新

**GCN 是特例**：$\psi$ 是线性变换，$\bigoplus$ 是归一化加权求和。

### 3.2 谱视角下的消息传递

消息传递相当于**图上的低通滤波**——平滑节点信号，让邻居趋同。

**过平滑问题**：堆太多层 GNN，所有节点表示趋同（信息被过度平滑）。这与谱分析里"反复低通滤波会抹平高频"一致。

## 四、工程实现层

### 4.1 GCN 层

```python
import torch
import torch.nn as nn

class GCNLayer(nn.Module):
    def __init__(self, in_dim, out_dim):
        super().__init__()
        self.W = nn.Linear(in_dim, out_dim)
    def forward(self, x, A_hat):
        # A_hat = D̃^{-1/2}(A+I)D̃^{-1/2} 预先算好
        return self.W(A_hat @ x)
```

### 4.2 归一化邻接矩阵

```python
def normalize_adj(A):
    # A: (n, n) 邻接矩阵
    A_tilde = A + torch.eye(A.size(0))
    D = A_tilde.sum(dim=1)
    D_inv_sqrt = D.pow(-0.5)
    return D_inv_sqrt.view(-1, 1) * A_tilde * D_inv_sqrt.view(1, -1)
```

### 4.3 完整 GCN

```python
class GCN(nn.Module):
    def __init__(self, in_dim, hidden, out_dim):
        super().__init__()
        self.l1 = GCNLayer(in_dim, hidden)
        self.l2 = GCNLayer(hidden, out_dim)
    def forward(self, x, A_hat):
        h = torch.relu(self.l1(x, A_hat))
        return self.l2(h, A_hat)
```

## 六、直觉总结

```
图拉普拉斯 L = D - A: 度量图上信号光滑度
L 的特征向量 = 图上的"正弦波"(频率从低到高)
谱滤波 = 在频域乘滤波函数

GNN 消息传递 ≈ 图上低通滤波(平滑邻居)
堆太多层 ⇒ 过平滑(节点趋同)
```

**GNN = 图上的谱滤波器。** 数学是图拉普拉斯，工程是稀疏矩阵乘。
