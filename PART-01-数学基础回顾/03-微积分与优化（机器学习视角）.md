## 一、导数与偏导数：变化率

### 1.1 一元导数

$$f'(x) = \lim_{h\to 0}\frac{f(x+h)-f(x)}{h}$$

**几何**：切线斜率。**优化意义**：损失下降最快的方向。

### 1.2 偏导数

多元函数 $f(x_1, \ldots, x_n)$ 对 $x_i$ 的偏导 = 其他变量固定时 $f$ 对 $x_i$ 的变化率。

$$\frac{\partial f}{\partial x_i}$$

> 📖 偏导数：[[Mathematics-Universe/03-高等数学/04-多元微分学/多元微分学详解.md]]

### 1.3 梯度：所有偏导组成的向量

$$\nabla f = \left(\frac{\partial f}{\partial x_1}, \ldots, \frac{\partial f}{\partial x_n}\right)$$

**核心事实**：$\nabla f$ 指向 $f$ 上升最快的方向，$-\nabla f$ 是下降最快的方向。**这就是梯度下降的数学根据。**

## 二、链式法则：反向传播的灵魂

### 2.1 单变量

若 $L = L(g(f(x)))$：

$$\frac{dL}{dx} = \frac{dL}{dg}\cdot\frac{dg}{df}\cdot\frac{df}{dx}$$

### 2.2 多变量（矩阵形式）

$$\frac{\partial L}{\partial \mathbf{x}} = \frac{\partial L}{\partial \mathbf{y}} \cdot J_{\mathbf{y}\to\mathbf{x}}$$

**这就是反向传播**：损失对各层的梯度 = 上游梯度 × 本层雅可比矩阵。

## 三、Taylor 展开：优化的根据

### 3.1 一阶 Taylor

$$f(\mathbf{x} + \Delta\mathbf{x}) \approx f(\mathbf{x}) + \nabla f(\mathbf{x})^\top \Delta\mathbf{x}$$

**推论**：要让 $f$ 下降最多，$\Delta\mathbf{x}$ 取 $-\nabla f$ 方向（负梯度）。这是梯度下降的理论根基。

### 3.2 二阶 Taylor

$$f(\mathbf{x}+\Delta\mathbf{x}) \approx f(\mathbf{x}) + \nabla f^\top\Delta\mathbf{x} + \frac{1}{2}\Delta\mathbf{x}^\top H \Delta\mathbf{x}$$

$H$ 是 Hessian（二阶导矩阵）。**Newton 法**解 $\nabla f + H\Delta\mathbf{x} = 0$ 得 $\Delta\mathbf{x} = -H^{-1}\nabla f$。

> 📖 二阶优化：[[轴线C/02-Taylor展开→二阶优化]]

## 四、梯度下降与凸性

### 4.1 梯度下降

$$\mathbf{x}_{t+1} = \mathbf{x}_t - \eta \nabla f(\mathbf{x}_t)$$

### 4.2 凸函数的好性质

$f$ 凸 $\Leftrightarrow$ $H \succeq 0$（Hessian 半正定）。凸函数**局部极小 = 全局最小**，梯度下降收敛有保证。

**深度学习的难处**：神经网络损失**非凸**，有大量鞍点和局部极小。但实践仍能找到好解。

> 📖 优化算法：[[PART-02/03-优化算法]]

## 五、自动微分：不用手算导数

### 5.1 符号 vs 数值 vs 自动

- **符号微分**：推导公式（容易表达式爆炸）
- **数值微分**：$(f(x+h)-f(x))/h$（精度差）
- **自动微分（AD）**：分解成基本运算，用链式法则精确算（深度学习用这个）

### 5.2 PyTorch autograd

```python
import torch
x = torch.tensor(2.0, requires_grad=True)
y = x ** 2 + 3 * x + 1
y.backward()         # 自动算 dy/dx
print(x.grad)        # tensor(7.)  = 2*2+3
```

## 七、直觉总结

```
导数: 变化率; 梯度: 所有偏导组成的向量(最快上升方向)
链式法则: 复合函数的导数 = 各环节导数相乘 = 反向传播
Taylor: 一阶⇒梯度下降, 二阶⇒Newton法
自动微分: 让计算机用链式法则精确算梯度
```

**微积分在深度学习里就一件事：算梯度、沿梯度下山。** autograd 让这一切自动化。
