# 函数空间与逼近论（机器学习视角）

> 神经网络本质是个**函数**：输入→输出。逼近论问：什么样的函数能用什么基去逼近到任意精度？通用近似定理就是现代版逼近论。本文只保留工程直觉，去掉 Banach/Sobolev 空间的纯数学。

**难度**：[进阶]（需要函数 + 极限 + 一点泛函直觉）

> 本文是函数空间视角的 ML 裁剪版，详细推导见 [[轴线D/01-Weierstrass逼近→通用近似定理]]。

## 一、函数空间：把函数当点

### 1.1 函数空间的概念

把满足某些性质的函数全体看作一个"空间"：
- $C[a,b]$：连续函数空间
- $L^2$：平方可积函数空间
- Sobolev 空间 $H^k$：带导数可积性的函数空间

**意义**：函数空间里可以定义**范数（距离）**、**内积**、**正交**——把函数当向量处理。

### 1.2 范数：衡量函数"大小"

$$\|f\|_\infty = \sup_x |f(x)|, \quad \|f\|_2 = \left(\int |f|^2\right)^{1/2}$$

**逼近的目标**：找简单函数 $g$ 让 $\|f - g\|$ 小。

---

## 二、函数逼近：找一组基

### 2.1 用基函数线性组合逼近

任何（足够好的）函数都能展开成基函数的叠加：

$$f(x) \approx \sum_i c_i \phi_i(x)$$

| 基 $\phi_i$ | 名称 | 应用 |
|------------|------|------|
| $1, x, x^2, \ldots$ | 多项式 | Taylor, Weierstrass |
| $\sin nx, \cos nx$ | 三角函数 | Fourier 分析 |
| $\sigma(w^\top x + b)$ | 神经元 | 神经网络 |

**通用近似定理**：sigmoid/ReLU 神经元的线性组合能逼近任何连续函数——神经网络是一种**新的基函数**。

> 📖 通用近似：[[轴线D/01-Weierstrass逼近→通用近似定理]]

---

## 三、稀疏与低维：好的表示

### 3.1 好基的特征

- **正交**：系数互不影响，数值稳定
- **稀疏**：少数几个基就能很好逼近（如小波压缩）
- **自适应**：基随数据变（神经网络学的就是数据自适应的基）

### 3.2 深度学习的表示学习

传统逼近用**固定基**（多项式、三角函数）。神经网络用**学出来的基**（隐藏单元）——这是它能超越传统方法的关键。

---

## 四、工程直觉

```python
# 神经网络 = 学出来的基函数逼近器
model = nn.Sequential(
    nn.Linear(1, 64), nn.ReLU(),   # 64 个学出来的"基函数"
    nn.Linear(64, 1)               # 线性组合
)
# 宽度 = 基的数量; 深度 = 复合层次
```

**宽度 vs 深度**：某些函数单层要指数多基，多层线性多就够——这就是"深"的数学理由。

## 六、直觉总结

```
函数空间: 把函数当点, 可定义距离/范数/正交
逼近论:   用基函数的线性组合近似任意函数
神经网络: 用"学出来的神经元"做基, 是现代逼近论
深度优势: 某些函数深网络用指数少参数(对比单层)
```

**神经网络 = 自适应基的函数逼近器。** 这是它能工作的理论基础。
