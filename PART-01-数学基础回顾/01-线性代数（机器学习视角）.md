# 线性代数（机器学习视角）

> 深度学习里几乎所有的"运算"都是**矩阵乘法**：一层网络、一次注意力、一次卷积，本质都是 $W\mathbf{x}$。本文不证明线代定理，只保留**深度学习真正用到的那部分**——向量空间、矩阵作为线性映射、特征值与 SVD、范数。

**难度**：[基础]（需要高中向量概念）

> 本文是 [[Mathematics-Universe/04-线性代数/线性代数全貌.md]] 的 ML 裁剪版，去掉纯数学的证明细节，只留工程必需。

## 一、向量与向量空间：数据的基本单位

### 1.1 向量 = 一列数

神经网络里，一个样本、一个特征、一组权重，都是向量 $\mathbf{x} \in \mathbb{R}^n$。

**关键运算**：
- 加法 $\mathbf{x} + \mathbf{y}$（叠加信号）
- 数乘 $\alpha \mathbf{x}$（缩放）
- 内积 $\mathbf{x}^\top \mathbf{y} = \sum_i x_i y_i$（相似度）
- 范数 $\|\mathbf{x}\|$（大小）

> 📖 向量空间：[[Mathematics-Universe/04-线性代数/03-向量与向量空间/向量与向量空间详解.md]]

### 1.2 内积的几何

$$\mathbf{x}^\top \mathbf{y} = \|\mathbf{x}\|\|\mathbf{y}\|\cos\theta$$

- $\theta = 0$（同向）：内积最大，**最相似**
- $\theta = 90°$（正交）：内积为 0，**不相关**
- $\theta = 180°$（反向）：内积最小（负），**最相反**

**这就是为什么注意力用 $Q K^\top$、对比学习用余弦相似度**——内积衡量相似性。

### 1.3 范数：衡量大小

| 范数 | 公式 | 用途 |
|------|------|------|
| L2 | $\sqrt{\sum x_i^2}$ | 欧氏距离、权重衰减 |
| L1 | $\sum \|x_i\|$ | 稀疏、L1 正则 |
| L∞ | $\max \|x_i\|$ | 一致逼近 |

## 二、矩阵 = 线性映射

### 2.1 核心视角转换

传统线代把矩阵当"数表"。**深度学习把矩阵当线性映射**：$W: \mathbb{R}^n \to \mathbb{R}^m$，$\mathbf{x} \mapsto W\mathbf{x}$。

**神经网络的一层就是**：仿射映射 $W\mathbf{x} + \mathbf{b}$ + 非线性。

### 2.2 矩阵乘法 = 复合映射

$W_2(W_1 \mathbf{x}) = (W_2 W_1)\mathbf{x}$。**矩阵相乘 = 线性映射的复合**。

### 2.3 转置 $W^\top$：对偶映射

$W^\top$ 是 $W$ 的对偶。反向传播里梯度"往回传"用的就是 $W^\top$：

$$\bar{\mathbf{h}}_{l-1} = W_l^\top \bar{\mathbf{z}}_l$$

> 📖 反向传播里的转置：[[PART-02/02-反向传播]]

## 三、特征值与特征向量：抓住主方向

### 3.1 定义

若 $W\mathbf{v} = \lambda \mathbf{v}$（$\mathbf{v} \neq 0$），则 $\mathbf{v}$ 是特征向量，$\lambda$ 是特征值。

**几何**：$W$ 作用在 $\mathbf{v}$ 上只缩放、不改变方向。

### 3.2 深度学习里的应用

- **PCA**：协方差矩阵的大特征值方向 = 数据主要变化方向
- **梯度爆炸/消失**：权重矩阵的最大特征值 $>1$ 爆炸，$<1$ 消失
- **谱归一化**：限制 $\|W\|_2 = \sigma_{\max}$ 稳定训练

> 📖 特征值：[[Mathematics-Universe/04-线性代数/05-特征值与特征向量/特征值与特征向量详解.md]]

## 四、SVD：万能分解

### 4.1 奇异值分解

任何矩阵 $W \in \mathbb{R}^{m\times n}$ 都能分解：

$$W = U \Sigma V^\top$$

- $U, V$ 正交矩阵（旋转）
- $\Sigma$ 对角矩阵（缩放，对角元 = 奇异值）

**几何**：任何线性变换 = 旋转 + 缩放 + 旋转。

### 4.2 低秩近似

保留前 $k$ 个大奇异值：

$$W_k = U_k \Sigma_k V_k^\top \approx W$$

**应用**：
- 模型压缩（用 $W_k$ 替代 $W$，参数从 $mn$ 降到 $k(m+n)$）
- PCA（数据矩阵的 SVD）
- LoRA（低秩适配微调大模型）

> 📖 SVD 与 PCA：[[轴线A/01-特征值分解→PCA→自编码器]]

## 五、张量：深度学习的多维数组

PyTorch 里数据是**张量**（多维数组）。向量是 1 维张量，矩阵是 2 维张量，批量图像是 4 维（batch, channel, height, width）。

```python
import torch
# 形状与广播
x = torch.randn(32, 784)          # batch=32 的向量
W = torch.randn(784, 128)         # 权重矩阵
h = x @ W                         # (32, 128) 矩阵乘法
# @ 是 matmul, 自动处理批量维度
```

## 七、直觉总结

```
向量: 数据的基本单位(样本/特征/权重)
矩阵: 线性映射(一层网络、一次变换)
内积: 相似度(注意力、对比学习)
特征值/SVD: 抓主方向(PCA、压缩、LoRA)
范数: 大小+正则(L2权重衰减、L1稀疏)
```

**深度学习 = 一连串矩阵乘法 + 非线性。** 线代是它的母语。
