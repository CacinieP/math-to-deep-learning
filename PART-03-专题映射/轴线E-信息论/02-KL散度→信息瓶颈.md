# KL 散度 → 信息瓶颈

> KL 散度衡量两个分布的差异：$D_{\text{KL}}(p\|q) = \sum p\log\frac{p}{q}$。在深度学习里，它是变分推断的核心（VAE 的损失项）、知识蒸馏的目标、信息瓶颈理论的基石——后者声称：好的表示应该**最大化与标签的信息，最小化与输入冗余信息**，整个过程用互信息（KL 的期望形式）刻画。

**难度**：[进阶]（需要 KL 散度 + 互信息 + 变分推断直觉）

## 一、纯数学版：从 KL 到互信息

### 1.1 KL 散度的性质回顾

$$D_{\text{KL}}(p \| q) = \sum_x p(x) \log \frac{p(x)}{q(x)} = \mathbb{E}_{x \sim p}\!\left[\log \frac{p(x)}{q(x)}\right]$$

- 非负：$D_{\text{KL}} \geq 0$，等号当 $p = q$（Gibbs 不等式）
- 非对称：$D_{\text{KL}}(p\|q) \neq D_{\text{KL}}(q\|p)$
- 凸性：对 $(p, q)$ 联合凸

> 📖 KL 散度：[[01-熵→交叉熵损失]]

### 1.2 互信息 = KL 的期望形式

两个随机变量 $X, Y$ 的互信息：

$$I(X; Y) = D_{\text{KL}}(p_{X,Y} \| p_X \otimes p_Y) = \mathbb{E}_{X,Y}\!\left[\log \frac{p(X,Y)}{p(X)p(Y)}\right]$$

**直觉**：$I(X;Y)$ 度量"知道 $X$ 能减少多少关于 $Y$ 的不确定性"。等价于：

$$I(X; Y) = H(Y) - H(Y|X) = H(X) - H(X|Y)$$

- $X, Y$ 独立：$I = 0$
- $X = Y$：$I = H(X)$（完全相关）

### 1.3 数据处理不等

若 $X \to Z \to Y$ 构成马尔可夫链（$Z$ 是从 $X$ 算出的表示，$Y$ 是标签），则：

$$I(X; Y) \geq I(Z; Y)$$

**含义**：经过表示 $Z$ 后，关于标签 $Y$ 的信息**只会减少，不会增加**。好的表示要让这个减少**尽量小**——尽量保留预测标签所需的信息。

## 二、信息瓶颈理论（Tishby, 2000）

### 2.1 核心目标

找一个表示 $Z = f(X)$，同时满足：
1. **压缩**：$Z$ 尽量"忘记"输入 $X$ 的细节 → 最小化 $I(X; Z)$
2. **预测**：$Z$ 尽量保留预测 $Y$ 的信息 → 最大化 $I(Z; Y)$

形式化为约束优化：

$$\min_{Z} \underbrace{I(X; Z)}_{\text{压缩}} - \beta \underbrace{I(Z; Y)}_{\text{预测}}$$

$\beta$ 控制压缩与预测的权衡。

### 2.2 信息瓶颈的几何图示

```
X (输入, 高维)
 │
 │  压缩: 丢掉与 Y 无关的细节
 ▼
Z (表示)        I(X;Z) 小(压缩)
 │              I(Z;Y) 大(预测)
 │  保留预测信息
 ▼
Y (标签)
```

### 2.3 与交叉熵损失的关系

普通交叉熵损失只最大化 $I(Z; Y)$（让表示预测标签），不约束 $I(X; Z)$。信息瓶颈额外要求压缩——这等价于一种**信息论正则化**，能提升泛化。

## 三、KL 散度在变分推断中的角色

### 3.1 VAE 的损失（ELBO）

变分自编码器假设隐变量 $\mathbf{z}$ 有先验 $p(\mathbf{z})$（通常标准正态），用神经网络学推断网络 $q_\phi(\mathbf{z}|x)$。最大化证据下界（ELBO）：

$$\mathcal{L}_{\text{ELBO}} = \underbrace{\mathbb{E}_{q}\!\left[\log p_\theta(x|\mathbf{z})\right]}_{\text{重建项}} - \underbrace{D_{\text{KL}}(q_\phi(\mathbf{z}|x) \| p(\mathbf{z}))}_{\text{KL 项}}$$

**KL 项的作用**：让推断网络的后验 $q_\phi$ 不要离先验 $p(\mathbf{z})$ 太远——这是**信息瓶颈的实例**：压缩隐表示到先验附近。

### 3.2 高斯情形的 KL 闭式解

若 $q = \mathcal{N}(\mu, \sigma^2)$，$p = \mathcal{N}(0, 1)$：

$$D_{\text{KL}}(q \| p) = \frac{1}{2}\left(\mu^2 + \sigma^2 - \log\sigma^2 - 1\right)$$

这就是 VAE 里 KL 项的具体形式。

> 📖 VAE 与正态分布：[[轴线B/01-Bayes定理→贝叶斯神经网络]]

## 四、知识蒸馏：KL 作为教学目标

### 4.1 软标签蒸馏

Teacher 模型输出软分布 $q^T$（带温度 $T$ 的 softmax）。Student 学：

$$\mathcal{L} = (1-\alpha) \cdot H(y, q^S) + \alpha \cdot T^2 \cdot D_{\text{KL}}(q^T \| q^S)$$

**KL 项**：逼 Student 模仿 Teacher 的**类间关系**（软标签的熵提供了相似度结构）。

### 4.2 为什么 KL 比纯交叉熵好

软标签 $q^T$ 不是 one-hot，熵 $>0$，携带"类间相似度"。Student 学到的不只是"是哪类"，还有"像哪类"——泛化更好。

## 五、工程实现层：PyTorch 实现

### 5.1 VAE 的 KL 项

```python
import torch

def vae_kl(mu, logvar):
    # q=N(μ,σ²), p=N(0,1) 的 KL 散度闭式解
    return -0.5 * torch.sum(1 + logvar - mu.pow(2) - logvar.exp(), dim=1).mean()

# VAE 训练
mu, logvar = encoder(x)
z = mu + torch.randn_like(mu) * (0.5 * logvar).exp()  # 重参数化
recon = decoder(z)
loss = recon_loss(recon, x) + vae_kl(mu, logvar)
```

### 5.2 知识蒸馏

```python
import torch.nn.functional as F

def distillation_loss(student_logits, teacher_logits, labels, T=4, alpha=0.7):
    soft = F.kl_div(
        F.log_softmax(student_logits / T, dim=1),
        F.softmax(teacher_logits / T, dim=1),
        reduction='batchmean'
    ) * (T * T)
    hard = F.cross_entropy(student_logits, labels)
    return (1 - alpha) * hard + alpha * soft
```

### 5.3 互信息的神经估计（MINE）

```python
# 用神经网络估计 I(X;Z) = E[log f(x,z)] - log E[e^f(x,z')]
# f 是可学习的判别网络, 区分联合分布与边缘乘积
def mine_loss(joint_scores, marginal_scores):
    return -(joint_scores.mean() - torch.logsumexp(marginal_scores, dim=0))
```

## 六、信息瓶颈能解释什么

### 6.1 泛化的信息论界限

理论上，泛化误差与 $I(X; Z)$（表示记住多少训练数据）有关。**压缩越好（$I(X;Z)$ 越小），泛化越好**。这给"深度网络为什么泛化"提供了信息论解释。

### 6.2 深度学习训练两阶段（Tishby 假说）

经验观察：训练初期网络**快速降低训练误差**（拟合阶段），后期**压缩表示**（$I(X;Z)$ 下降，泛化提升）。这个"压缩阶段"是否真实存在仍有争议。

### 6.3 表示解耦

好的表示里，不同维度应**统计独立**——即 $I(Z_i; Z_j)$ 小。这是 $\beta$-VAE 等解耦表示学习的信息论目标。

## 八、直觉总结

```
KL散度: 两个分布的差异(≥0, 越小越像)
互信息: 知道 X 能减少多少 Y 的不确定性

信息瓶颈: 好表示 = 少记输入(I(X;Z)小) + 多保标签(I(Z;Y)大)
         类比: 考试复习只记考点, 不抄整本书

VAE: KL 项把隐表示压向先验(信息瓶颈实例)
蒸馏: KL 让 student 模仿 teacher 的类间相似度
```

**KL 散度是深度学习里"约束分布形状"的通用工具。** 数学是信息论，工程是各种损失项的组合。
