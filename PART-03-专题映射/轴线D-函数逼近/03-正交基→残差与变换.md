# 正交基 → 残差与变换

> 正交基让向量能在不同表示间无损切换：$\mathbf{v} = \sum_i \langle \mathbf{v}, \mathbf{e}_i\rangle \mathbf{e}_i$。深度学习里，残差连接 $\mathbf{h}_{l+1} = \mathbf{h}_l + f_l(\mathbf{h}_l)$ 本质上是在做**恒等基 + 学到的修正**——网络只在残差子空间里学增量，这既缓解了梯度消失，又把"全量学习"降维成"差量学习"。

**难度**：[进阶]（需要正交基 + 投影 + 残差网络经验）

## 一、纯数学版：正交基与投影

### 1.1 正交基的定义

内积空间 $V$ 中，一组基 $\{\mathbf{e}_i\}$ 若满足：

$$\langle \mathbf{e}_i, \mathbf{e}_j\rangle = \delta_{ij}$$

称为**标准正交基**（ONB）。

**例子**：
- $\mathbb{R}^n$ 的标准基 $\{\mathbf{e}_1, \ldots, \mathbf{e}_n\}$
- Fourier 基 $\{e^{inx}\}$（在 $L^2$ 中）
- Haar 小波基、Legendre 多项式

> 📖 正交：[[Mathematics-Universe/04-线性代数/05-特征值与特征向量/特征值与特征向量详解.md]]

### 1.2 正交投影与展开

任何 $\mathbf{v} \in V$ 可唯一展开：

$$\mathbf{v} = \sum_i \langle \mathbf{v}, \mathbf{e}_i\rangle \mathbf{e}_i$$

**展开系数 = 投影长度**。正交基的价值：表示与重建都**最简单、最稳定**（数值不放大误差）。

### 1.3 正交变换保长度保内积

正交矩阵 $Q$（$Q^\top Q = I$）满足：

$$\|Q\mathbf{v}\| = \|\mathbf{v}\|, \quad \langle Q\mathbf{u}, Q\mathbf{v}\rangle = \langle \mathbf{u}, \mathbf{v}\rangle$$

**几何**：旋转、反射、置换——刚体变换，不改变距离。这在残差网络分析里极重要。

### 1.4 完备性与 Parseval 等式

若 $\{\mathbf{e}_i\}$ 完备（构成整个空间），则：

$$\|\mathbf{v}\|^2 = \sum_i |\langle \mathbf{v}, \mathbf{e}_i\rangle|^2 \quad \text{(Parseval)}$$

**能量守恒**：信号在时域和频域的能量相等。

## 二、残差连接：恒等基 + 修正

### 2.1 残差块的结构

ResNet 的核心模块：

$$\mathbf{h}_{l+1} = \mathbf{h}_l + f_l(\mathbf{h}_l)$$

输出 = **输入（恒等）** + **学到的残差 $f_l$**。

### 2.2 正交基视角：在恒等方向上做小扰动

把恒等映射 $\mathbf{h} \mapsto \mathbf{h}$ 看作"基方向"，$f_l(\mathbf{h})$ 看作"修正方向"：

$$\text{输出} = \underbrace{\mathbf{h}}_{\text{基(已知)}} + \underbrace{f_l(\mathbf{h})}_{\text{残差(要学)}}$$

**对比无残差**：$\mathbf{h}_{l+1} = g_l(\mathbf{h}_l)$ 要让 $g_l$ 同时学"保留信息"和"变换信息"，负担重。残差把"保留"交给恒等，让 $f_l$ 只学"变换"。

### 2.3 从优化视角：梯度的高速公路

反向传播时（链式法则）：

$$\frac{\partial L}{\partial \mathbf{h}_l} = \frac{\partial L}{\partial \mathbf{h}_{l+1}} \cdot \left(I + \frac{\partial f_l}{\partial \mathbf{h}_l}\right)$$

**关键项 $I$**：无论 $\partial f_l/\partial \mathbf{h}_l$ 多小，$I$ 保证了梯度能**无损回流**。这是深网络能训练的数学根基。

> 📖 链式法则：[[轴线C/01-梯度→反向传播]]

### 2.4 残差 = 显式恒等映射

实验发现：移除残差的"加法"（只用 $f_l$），深网络立刻退化；加上后，即使上百层也能训练。**残差让恒等映射成为网络的"默认行为"**，学习只需修正。

## 三、正交性与网络稳定性的深层联系

### 3.1 正交初始化为何有效

若每层权重 $W_l$ 初始化为近似正交，则信号前向传播：

$$\|W_l \mathbf{h}\| \approx \|\mathbf{h}\|$$

**信号能量守恒**——既不爆炸也不消失。这就是 Xavier/He 初始化的数学根据（让权重方差满足正交性近似）。

> 📖 初始化与方差：[[轴线B/03-正态分布→Xavier-He初始化]]

### 3.2 谱范数与 Lipschitz 约束

矩阵的谱范数 $\|W\|_2 = \sigma_{\max}(W)$（最大奇异值）。若 $\|W\|_2 \leq 1$，则 $W$ 是 Lipschitz 常数 $\leq 1$ 的映射——**压缩映射**。

**应用**：
- GAN 里谱归一化判别器（让 Lipschitz 有界，稳定训练）
- 深网络里限制每层谱范数，缓解梯度爆炸

### 3.3 正交权重机器学习研究

一些工作直接**约束权重正交**（$W^\top W = I$），理论上保证信号完美传递。但实践上正交约束太强，牺牲表达力，多用于特定场景（RNN 缓解梯度消失）。

## 四、Transformer 里的正交与残差

### 4.1 残差是 Transformer 的骨架

每个 Transformer 层都是：

$$\mathbf{h}_{l+1} = \mathbf{h}_l + \text{SubLayer}(\text{Norm}(\mathbf{h}_l))$$

**子层**（注意力、FFN）都包在残差里。没有残差，Transformer 根本训不动。

### 4.2 Layer Norm 的正交化作用

Layer Norm 把每个样本的特征归一化到均值 0、方差 1——近似做**正交化**（去除特征的尺度差异），让梯度各方向更均衡。

### 4.3 注意力残差 = 基变换 + 恒等保留

注意力做基变换（用学到的值向量做展开），残差保留恒等。两者结合 = "**在恒等基的补空间里做自适应展开**"。

## 五、工程实现层：PyTorch 残差与正交

### 5.1 残差块

```python
import torch.nn as nn

class ResidualBlock(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(dim, dim), nn.ReLU(), nn.Linear(dim, dim)
        )
    def forward(self, x):
        return x + self.net(x)   # 恒等 + 残差

# 100 层也能训练
model = nn.Sequential(*[ResidualBlock(128) for _ in range(100)])
```

### 5.2 正交初始化

```python
import torch.nn.init as init

linear = nn.Linear(512, 512)
init.orthogonal_(linear.weight)    # 正交初始化权重
# 信号前向时能量近似守恒, 缓解深网络梯度问题
```

### 5.3 谱归一化（GAN 用）

```python
import torch.nn.utils.spectral_norm as sn

# 用谱归一化包裹线性层, 强制 ‖W‖₂ ≤ 1
discriminator = sn(nn.Linear(128, 1))
```

### 5.4 Transformer 层（残差 + Norm + Attention）

```python
class TransformerLayer(nn.Module):
    def __init__(self, d=512, h=8):
        super().__init__()
        self.attn = MultiHeadAttention(d, h)
        self.norm1 = nn.LayerNorm(d)
        self.ffn = nn.Sequential(nn.Linear(d, d*4), nn.GELU(), nn.Linear(d*4, d))
        self.norm2 = nn.LayerNorm(d)
    def forward(self, x):
        x = x + self.attn(self.norm1(x))   # 残差1: 注意力
        x = x + self.ffn(self.norm2(x))    # 残差2: FFN
        return x
```

## 七、直觉总结

```
正交基: 让表示最稳定, 信息无损切换
残差:    输出 = 输入(恒等基) + 修正(要学的)
         把"保留信息"交给恒等, 把"变换信息"交给网络
梯度:    残差提供 I 这条高速公路, 深网络才训得动

正交初始化: 信号前向能量守恒, 防爆炸/消失
谱归一化:   强制 ‖W‖≤1, 压缩映射, 稳定训练
```

**残差 = 用恒等做"已知基"，让网络只在补空间里学增量。** 数学是正交投影，工程是加法与初始化。
