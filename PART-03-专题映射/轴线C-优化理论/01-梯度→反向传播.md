# 梯度 → 反向传播

> 反向传播的本质就是链式法则的矩阵形式。从单变量的 $\frac{df}{dx} = \frac{df}{dy} \cdot \frac{dy}{dx}$ 到百万参数的神经网络，中间只差一个 Jacobi 矩阵。

**难度**：[标准]（需要微积分 + 矩阵运算 + 了解神经网络前向传播）

## 一、纯数学版

### 1.1 单变量链式法则

$$\frac{df}{dx} = \frac{df}{dy} \cdot \frac{dy}{dx}$$

如果 $f = g \circ h$（复合函数）：
$$(g \circ h)'(x) = g'(h(x)) \cdot h'(x)$$

多层链式法则：
$$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial y} \cdot \frac{\partial y}{\partial x}$$

**这是反向传播的灵魂**——所有中间梯度都通过链式法则从后往前传播。

> 📖 链式法则：[[Mathematics-Universe/03-高等数学/02-一元微分学/一元微分学详解.md#22-经典案例证明-ln1x-x]]

### 1.2 多变量链式法则（Jacobi 矩阵）

对于向量值函数 $\mathbf{y} = f(\mathbf{x})$，其中 $\mathbf{x} \in \mathbb{R}^n, \mathbf{y} \in \mathbb{R}^m$：

$$\frac{\partial \mathbf{y}}{\partial \mathbf{x}} = J_f(\mathbf{x}) \in \mathbb{R}^{m \times n}$$

其中 $J_{ij} = \frac{\partial y_i}{\partial x_j}$。

**链式法则的矩阵形式**：

若 $\mathbf{z} = g(\mathbf{y})$，$\mathbf{y} = f(\mathbf{x})$，则：

$$\frac{\partial z}{\partial \mathbf{x}} = \frac{\partial z}{\partial \mathbf{y}} \cdot \frac{\partial \mathbf{y}}{\partial \mathbf{x}}$$

这里：
- $\frac{\partial z}{\partial \mathbf{y}}$ 是行向量 $(1 \times m)$
- $\frac{\partial \mathbf{y}}{\partial \mathbf{x}}$ 是 Jacobi 矩阵 $(m \times n)$
- 结果 $\frac{\partial z}{\partial \mathbf{x}}$ 是行向量 $(1 \times n)$

> 📖 Jacobi 矩阵：[[Mathematics-Universe/03-高等数学/04-多元微分学/多元微分学详解.md#42-Jacobi矩阵]]

## 二、计算图：把神经网络展开成数学表达式

### 2.1 一个简单网络的计算图

考虑一个 2 层网络：

```
输入 x ∈ R^d
    │
    ▼
z₁ = W₁x + b₁      (W₁: h₁×d, b₁: h₁)
    │
    ▼
h₁ = σ(z₁)         (ReLU)
    │
    ▼
z₂ = W₂h₁ + b₂     (W₂: h₂×h₁, b₂: h₂)
    │
    ▼
h₂ = σ(z₂)         (ReLU)
    │
    ▼
ŷ = W₃h₂ + b₃      (W₃: C×h₂, b₃: C)
    │
    ▼
L = CrossEntropy(ŷ, y)
```

**展开成数学表达式**：

$$L = -\sum_{c=1}^C y_c \log\left(\frac{e^{\hat{y}_c}}{\sum_j e^{\hat{y}_j}}\right)$$

其中：
$$\hat{y} = W_3 \sigma(W_2 \sigma(W_1 x + b_1) + b_2) + b_3$$

### 2.2 计算图的有向无环图（DAG）

```
           x
           │
     ┌─────┴─────┐
     ▼           ▼
  W₁·x        b₁          ← 线性变换的两个部分
     │           │
     └─────┬─────┘
           ▼
         z₁ = W₁x + b₁
           │
           ▼
         h₁ = σ(z₁)       ← ReLU
           │
     ┌─────┴─────┐
     ▼           ▼
  W₂·h₁       b₂
     │           │
     └─────┬─────┘
           ▼
         z₂ = W₂h₁ + b₂
           │
           ▼
         h₂ = σ(z₂)
           │
     ┌─────┴─────┐
     ▼           ▼
  W₃·h₂       b₃
     │           │
     └─────┬─────┘
           ▼
         ŷ = logits
           │
           ▼
         L = loss(ŷ, y)
```

**每条边对应一个运算，每个节点是一个中间结果。反向传播就是在这个图上从后往前传梯度。**

## 三、反向传播：从后往前的链式法则

### 3.1 核心思想

**前向传播**：从输入到输出，逐层计算激活值
**反向传播**：从输出到输入，逐层计算梯度

$$\frac{\partial L}{\partial W_1} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial h_2} \cdot \frac{\partial h_2}{\partial z_2} \cdot \frac{\partial z_2}{\partial h_1} \cdot \frac{\partial h_1}{\partial z_1} \cdot \frac{\partial z_1}{\partial W_1}$$

**关键技巧**：不每次都从头算乘积，而是**缓存中间结果**（前向传播时已经算过）。

### 3.2 逐层反向传播的数学

以 $L \to \hat{y} \to z_3 \to h_2 \to z_2 \to h_1 \to z_1 \to W_1$ 为例：

**Step 1**：$\frac{\partial L}{\partial \hat{y}}$（交叉熵 + softmax 的梯度）

$$\frac{\partial L}{\partial \hat{y}_c} = \text{softmax}(\hat{y})_c - y_c$$

这是 softmax + 交叉熵的**简洁梯度公式**——两个函数复合的导数恰好抵消了复杂部分。

**Step 2**：$\frac{\partial L}{\partial W_3}$

$$\frac{\partial L}{\partial W_3} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial W_3} = \delta_3 \cdot h_2^T$$

其中 $\delta_3 = \frac{\partial L}{\partial \hat{y}}$ 是 **误差项**。

**Step 3**：$\frac{\partial L}{\partial h_2}$

$$\frac{\partial L}{\partial h_2} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial h_2} = \delta_3^T \cdot W_3 = W_3^T \delta_3$$

**Step 4**：$\frac{\partial L}{\partial z_2}$（通过激活函数的导数）

$$\frac{\partial L}{\partial z_2} = \frac{\partial L}{\partial h_2} \odot \sigma'(z_2) = \delta_3^T W_3 \odot \mathbb{1}_{z_2 > 0}$$

对于 ReLU：$\sigma'(z) = \mathbb{1}_{z > 0}$（>0 时导数为1，否则为0）

**Step 5**：$\frac{\partial L}{\partial W_2}$

$$\frac{\partial L}{\partial W_2} = \delta_2 \cdot h_1^T, \quad \delta_2 = \left(\frac{\partial L}{\partial z_2}\right)^T$$

**Step 6**：依此类推...

### 3.3 通用公式

对于第 $l$ 层：
$$z_l = W_l h_{l-1} + b_l, \quad h_l = \sigma(z_l)$$

**反向传播的通用递推**：

$$\delta_l = (W_{l+1}^T \delta_{l+1}) \odot \sigma'(z_l)$$

$$\frac{\partial L}{\partial W_l} = \delta_l h_{l-1}^T, \quad \frac{\partial L}{\partial b_l} = \delta_l$$

**这就是反向传播的完整数学**——只有两行递推公式。

## 四、计算图上的自动微分

### 4.1 前向模式 vs 反向模式

| 模式 | 计算方向 | 适用场景 | 复杂度 |
|------|---------|---------|-------|
| **前向模式** | $x \to y \to \cdots \to f$ | 输入维度 $n$ 小，输出维度 $m$ 大 | $O(n \cdot \text{ops})$ |
| **反向模式** | $\frac{\partial f}{\partial x}$ 从后往前 | 输入维度 $n$ 大，输出维度 $m$ 小 | $O(m \cdot \text{ops})$ |

神经网络的典型情况：
- 输入：$x \in \mathbb{R}^{784}$（MNIST 图像）
- 输出：$\hat{y} \in \mathbb{R}^{10}$（10类概率）
- Loss：$L$ 是标量

$n = 784 \gg m = 1$ → **必须用反向模式**（也就是反向传播）。

### 4.2 PyTorch 的 autograd 是怎么工作的

```python
# 你写的代码
x = torch.tensor([1.0, 2.0, 3.0], requires_grad=True)
W = torch.randn(4, 3, requires_grad=True)
z = W @ x           # 前向：计算 z = Wx
loss = z.sum()      # 标量损失
loss.backward()     # 反向：自动计算所有梯度

# PyTorch 内部做了什么：
# 1. 构建计算图（通过 Function.apply 包装每个运算）
# 2. 前向传播时记录中间结果
# 3. backward() 时从 loss 开始反向遍历计算图
# 4. 每个节点调用其 backward 函数（手工编码的梯度公式）
```

**每个 PyTorch 运算都有两个实现**：
- `forward(input)`：前向计算
- `backward(grad_output)`：反向传播的梯度公式

```python
# 以矩阵乘法为例
class MatMulFunction(torch.autograd.Function):
    @staticmethod
    def forward(ctx, A, B):
        ctx.save_for_backward(A, B)    # 缓存前向的输入
        return A @ B

    @staticmethod
    def backward(ctx, grad_output):
        A, B = ctx.saved_tensors
        # dL/dA = dL/dZ * dB^T
        grad_A = grad_output @ B.T
        # dL/dB = A^T * dL/dZ
        grad_B = A.T @ grad_output
        return grad_A, grad_B

# 使用时
z = MatMulFunction.apply(W, x)  # 等价于 W @ x
```

## 五、手动实现反向传播

### 5.1 最简化示例：2层 MLP 从零实现

```python
import torch

class TwoLayerNet:
    """从零实现的反向传播——不用 autograd"""
    def __init__(self, d_in, h, d_out):
        # 初始化
        self.W1 = torch.randn(d_in, h) * 0.1
        self.b1 = torch.zeros(h)
        self.W2 = torch.randn(h, d_out) * 0.1
        self.b2 = torch.zeros(d_out)

        # 缓存前向传播的中间结果（反向传播需要）
        self.cache = {}

    def forward(self, x):
        """前向传播"""
        # Layer 1
        z1 = x @ self.W1 + self.b1       # (batch, h)
        h1 = torch.relu(z1)              # (batch, h)

        # Layer 2
        z2 = h1 @ self.W2 + self.b2       # (batch, d_out)

        # 缓存（反向传播需要）
        self.cache = {'x': x, 'z1': z1, 'h1': h1, 'z2': z2}
        return z2

    def backward(self, grad_output):
        """反向传播——手动实现"""
        x, z1, h1, z2 = self.cache['x'], self.cache['z1'], self.cache['h1'], self.cache['z2']
        batch = x.shape[0]

        # dL/dW2 = h1^T @ dL/dz2
        self.grad_W2 = h1.T @ grad_output / batch
        # dL/db2 = sum(dL/dz2)
        self.grad_b2 = grad_output.sum(dim=0) / batch

        # dL/dh1 = dL/dz2 @ W2^T
        grad_h1 = grad_output @ self.W2.T
        # dL/dz1 = dL/dh1 * relu'(z1)
        grad_z1 = grad_h1 * (z1 > 0).float()  # ReLU 导数

        # dL/dW1 = x^T @ dL/dz1
        self.grad_W1 = x.T @ grad_z1 / batch
        # dL/db1 = sum(dL/dz1)
        self.grad_b1 = grad_z1.sum(dim=0) / batch

    def step(self, lr):
        """SGD 更新"""
        self.W1 -= lr * self.grad_W1
        self.b1 -= lr * self.grad_b1
        self.W2 -= lr * self.grad_W2
        self.b2 -= lr * self.grad_b2

# 训练
model = TwoLayerNet(d_in=10, h=32, d_out=2)
x = torch.randn(128, 10)
y = torch.randint(0, 2, (128,))

# 手动训练循环
for epoch in range(100):
    # 前向
    logits = model.forward(x)                          # (128, 2)

    # 损失（交叉熵）
    log_probs = torch.log_softmax(logits, dim=-1)
    loss = -log_probs[range(128), y].mean()

    # 反向（手动！）
    # dL/dlogits = softmax(logits) - one_hot(y)
    grad_logits = torch.softmax(logits, dim=-1)
    grad_logits[range(128), y] -= 1
    grad_logits /= 128
    model.backward(grad_logits)

    # 更新
    model.step(lr=0.1)

    if epoch % 20 == 0:
        print(f"Epoch {epoch}: loss = {loss.item():.4f}")
```

### 5.2 反向传播 vs 有限差分

验证梯度正确性的黄金标准：

$$\frac{\partial L}{\partial W_{ij}} \approx \frac{L(W + \epsilon e_{ij}) - L(W - \epsilon e_{ij})}{2\epsilon}$$

```python
def numerical_gradient(f, W, eps=1e-5):
    """数值梯度（用于验证）"""
    grad = torch.zeros_like(W)
    for i in range(W.shape[0]):
        for j in range(W.shape[1]):
            W_plus = W.clone()
            W_plus[i, j] += eps
            W_minus = W.clone()
            W_minus[i, j] -= eps
            grad[i, j] = (f(W_plus) - f(W_minus)) / (2 * eps)
    return grad

# 验证
analytical_grad = model.grad_W1
numerical_grad = numerical_gradient(
    lambda w: nn.functional.cross_entropy(
        model.forward_with_W1(x, w), y), model.W1
)
print(f"最大差异: {(analytical_grad - numerical_grad).abs().max():.2e}")
# 应该是 ~1e-7 级别，说明反向传播正确
```

## 六、计算图上的"魔法"

### 6.1 为什么反向传播比数值微分快

| 方法 | 计算量 | 对 100 万参数网络 |
|------|--------|-----------------|
| 数值微分（前向差分） | $O(n)$ 次前向传播 | 100万次前向 |
| 数值微分（中心差分） | $O(n)$ 次前向传播 | 100万次前向 |
| **反向传播** | $O(1)$ 次前向 + $O(1)$ 次反向 | **2次传播** |

反向传播的复杂度是**与参数数量无关的常数倍**——因为它在一次反向遍历中同时计算所有梯度。

### 6.2 计算图上的 Jacobi 矩阵

计算图本质上是在做 **Jacobian-vector product (JVP)** 和 **vector-Jacobian product (VJP)**：

```
前向模式（JVP）：                    反向模式（VJP）：

    x ──[f]──→ y                       L ←——[f]—— y
    │           │                       ↑           ↑
  dx           dy                     dL          dy
    │           │                       ↑           ↑
  J·dx        y'=f(x)                  J^T·dL    f'(x)

J: Jacobian 矩阵 (m×n)
J·dx: 前向传播的切线（在 x 方向上 f 的变化率）
J^T·dL: 反向传播的伴随（损失对各输入的敏感度）
```

**自动微分框架（PyTorch/TensorFlow/JAX）的核心就是高效计算 VJP**。

## 七、进阶：计算图上的"旁路"

### 7.1 非可微操作的梯度

有些操作没有解析导数（如 ReLU 在 0 处的导数、$\max$ 操作、$\text{argmax}$）。实践中用**次梯度**或**直通估计器（Straight-Through Estimator）**：

```python
# ReLU 在 0 处的次梯度
class CustomReLU(torch.autograd.Function):
    @staticmethod
    def forward(ctx, x):
        ctx.save_for_backward(x)
        return torch.relu(x)

    @staticmethod
    def backward(ctx, grad_output):
        x, = ctx.saved_tensors
        # 在 0 处用 1 代替（次梯度）
        grad_input = grad_output.clone()
        grad_input[x < 0] = 0
        grad_input[x == 0] = 1  # 次梯度约定
        return grad_input

# 直通估计器：把离散操作当作恒等映射的反向
class StraightThrough(torch.autograd.Function):
    @staticmethod
    def forward(ctx, x):
        return (x > 0).float()  # 前向：硬阈值（二值化）

    @staticmethod
    def backward(ctx, grad_output):
        # 反向：当作恒等映射（梯度直通）
        return grad_output
```

### 7.2 二阶导数（Hessian-Vector Product）

完整 Hessian 矩阵对于百万参数网络不可行（$O(n^2)$ 内存）。但可以用 **Pearlmutter (1994) 的技巧**：

```python
# 不用显式构建 H = ∂²L/∂W²
# 而是计算 H·v（Hessian-vector product）在 O(n) 时间内

def hvp(loss, params, v):
    """
    Hessian-vector product: H @ v
    使用二阶自动微分或"正向-反向-正向"技巧
    """
    # 方法1: PyTorch 二阶自动微分
    grad = torch.autograd.grad(loss, params, create_graph=True)
    grad_dot_v = sum((g * v_).sum() for g, v_ in zip(grad, v))
    hvp = torch.autograd.grad(grad_dot_v, params)
    return hvp
```

## 九、直觉总结

**一句话理解反向传播**：

```
前向: x → z₁ → h₁ → z₂ → h₂ → ŷ → L
       每步: "算出一个值"

反向: L → ŷ → h₂ → z₂ → h₁ → z₁ → x
       每步: "这个变化对前面有多大影响？"

链式法则就是答案: ∂L/∂W = ∂L/∂ŷ · ∂ŷ/∂h₂ · ∂h₂/∂z₂ · ∂z₂/∂W

"损失对 W 的影响 = 损失对输出的影响 × 输出对 W 的影响"
```

**计算图 + Jacobi 矩阵 + 链式法则 = 反向传播**。这就是全部。
